safeContact
===========

Safe Contact is a FREE module for Joomla! 1.5, 1.6, 1.7, 2.5 and 3.0 built on basis of Rapid Contact module (http://mavrosxristoforos.com/joomla-extensions/free/rapid-contact). 

This module is easy configureable and gives good SPAM protection, since it includes reCaptcha protection which is optional.

Also reCaptcha theme used here is custom, in order to fit image into the tight space too.
